# MakerWorld Scoreboard

A desktop companion for tracking MakerWorld statistics in near real time.

## Features

- Live MakerWorld statistics
- Historical snapshot storage
- Delta calculations
- Event generation
- Chrome extension collector
- FastAPI backend
- SQLite storage
